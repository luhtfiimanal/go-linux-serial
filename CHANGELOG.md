# Changelog

- Initial public release
- Linux-only, killable, low-latency serial port reader
- `SerialReader` with `ReadLine`, `ReadLinesLoop`, `WriteLine`, and `Close`
- PTY-based unit tests for reliability
- MIT License
- GoDoc polish and usage examples
